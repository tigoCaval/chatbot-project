# Chatbot
Exemplo de conversação humano x máquina (para fins didáticos).

# Requisitos
- Baixar projeto 
- Composer (instalar)
- PHP >= 7.3
- Executar comando ```composer update``` ou ```composer require tigo/chatterbot```
